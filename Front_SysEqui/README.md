# Front_SysEqui

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## Setup

```sh
npm install
```

## Configuration

The project is configured to use Vite's [Environment Variables](https://vitejs.dev/guide/env-and-mode.html#env-files) feature.

To use this feature, copy the `.env.example` file to a file named `.env` in the project root. This file will be loaded by default when running `npm run dev` or `npm run build`.

## Development

Once the project is created, you can start the development server:

```sh
npm run dev
```
